## 模块/包与commonJS规范
* 内置（系统自带，node安装好就已经存在了。）
* 第三方模块（被安装在node-modules文件夹中）
    * npm init
    * npm install 模块名  --save
    * npm uninstall 模块名
* 自定义（项目里独立的js文件，根据commonjs规范定义）
